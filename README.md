# UCSD Math 10B - Calculus II Website

A modern, aesthetic educational website for UCSD Math 10B, built with React, Vite, and Tailwind CSS.

## Features

-   **Comprehensive Curriculum**: Covers modules from Fundamentals of Integration to Differential Equations.
-   **Extensive Detail**: Detailed explanations, definitions, and examples for each topic.
-   **Math Rendering**: Beautiful LaTeX math rendering using KaTeX.
-   **Responsive Design**: Mobile-friendly layout with a collapsible sidebar.
-   **Clean Aesthetic**: Typography-focused design using Tailwind CSS Prose.

## Tech Stack

-   **Framework**: React + Vite
-   **Styling**: Tailwind CSS + Typography Plugin
-   **Routing**: React Router DOM
-   **Icons**: Lucide React
-   **Content**: React Markdown + Remark Math + Rehype KaTeX

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

3.  Build for production:
    ```bash
    npm run build
    ```
